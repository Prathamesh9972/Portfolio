## 🚀 Getting Started

1. Clone the repository
```bash
git clone https://github.com/your-username/portfolio.git
```

2. Navigate to the project directory
```bash
cd portfolio
```

3. Open `index.html` in your browser
```bash
# For macOS
open index.html

# For Linux
xdg-open index.html

# For Windows
start index.html
```
